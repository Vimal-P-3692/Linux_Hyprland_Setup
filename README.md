# Hyprland Setup:
  This is an hyprland setup. The hyprland setup was taken from the AvengeMedia/DankMaterialShell and the website link is https://danklinux.com/.

# Installition Process:
  git clone https://github.com/Vimal-P-3692/Linux_Hyprland_Setup.git  # clone this repo
  cd Linux_Hyprland_Setup/                                            # get inside the folder
  ./install.sh                                                        # run the script

  It will take some time to install the full setup.
